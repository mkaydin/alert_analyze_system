# T1059.001: Command and Scripting Interpreter — PowerShell

## Description
Adversaries may abuse PowerShell for execution, discovery, and lateral movement. PowerShell is a powerful scripting environment available by default on modern Windows systems. Attackers use it to download payloads, execute arbitrary code, enumerate systems, and bypass defenses.

## Detection
- PowerShell executed with encoded commands (`-EncodedCommand`, `-e`)
- PowerShell downloading content from the internet (`System.Net.WebClient`, `Invoke-WebRequest`)
- PowerShell reflection loading (`Assembly.Load`, `Invoke-Expression`)
- PowerShell writing to or reading from the registry
- Event ID 4103 (Module logging) and 4104 (Script block logging)
- Suspicious parameters like `-WindowStyle Hidden`, `-ExecutionPolicy Bypass`, `-NoProfile`

## Common Indicators
- Base64-encoded command lines
- Download cradles: `IEX (New-Object Net.WebClient).DownloadString('http://...')`
- In-memory execution: `Invoke-ReflectivePEInjection`, `Invoke-Mimikatz`
- Pipeline obfuscation, backtick characters, or variable substitution for evasion
